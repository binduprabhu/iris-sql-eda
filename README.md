# Iris SQL EDA (BigQuery)

Dataset: https://www.kaggle.com/datasets/uciml/iris

BigQuery table:
`iris-484612.irisdataset.iristable`

## What this project does
EDA in GoogleSQL:
- row count + missing values
- summary stats by species
- petal ranges by species
- correlations
- outlier candidates (p01/p99)
- quartiles (IQR) by species
